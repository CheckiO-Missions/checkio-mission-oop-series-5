5.1. Создайте класс ElectricCar, который наследует свойства от класса Car.

5.2. Измените метод __init__ класса ElectricCar так, чтобы он использовал super() для вызова метода __init__ класса Car.

5.3. Добавьте новый атрибут battery_capacity типа int в метод __init__ класса ElectricCar. Этот атрибут должен передаваться в качестве аргумента. Поместите этот аргумент перед brand и model, потому что аргументы без значений по умолчанию должны идти перед аргументами с такими значениями.

5.4. Создайте экземпляр my_electric_car класса ElectricCar, передав ему battery_capacity, brand, model со значениями 100, "Tesla", "Model 3" соответственно.
