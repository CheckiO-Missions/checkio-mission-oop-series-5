5.2. Proszę zmodyfikować metodę __init__ klasy ElectricCar tak, aby używała metody super() do wywołania metody __init__ klasy Car.

5.4. Proszę utworzyć instancję my_electric_car klasy ElectricCar, przekazując odpowiednio battery_capacity, brand, model z wartościami 100, "Tesla", "Model 3".
